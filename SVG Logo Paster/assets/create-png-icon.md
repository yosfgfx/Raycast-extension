# PNG Icon Required

This extension requires a PNG icon file named `vault-icon.png` in the assets folder.

The icon should be:
- Format: PNG
- Size: 256x256 pixels (recommended)
- Background: Transparent or solid color
- Content: Logo vault themed icon

To create the icon:
1. Convert the existing `vault-icon.svg` to PNG format
2. Save it as `assets/vault-icon.png`
3. Ensure it's 256x256 pixels

You can use online converters or tools like:
- Photoshop
- GIMP
- Online SVG to PNG converters
- ImageMagick: `convert vault-icon.svg -resize 256x256 vault-icon.png`
