# Icon Conversion Note

The extension requires a PNG icon format. Since we have an SVG icon, you would need to convert the `vault-icon.svg` to `vault-icon.png` using an image conversion tool.

For now, please manually convert the SVG file to PNG format (256x256 pixels recommended) and save it as `assets/vault-icon.png`.

You can use online tools like:
- https://svgtopng.com/
- https://cloudconvert.com/svg-to-png
- Or use ImageMagick: `convert vault-icon.svg vault-icon.png`
